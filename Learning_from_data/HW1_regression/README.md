# HW1 Regression - Learning From Data

This homework is an introduction to core supervised learning concepts and basic probability.

It includes:

- Part 1: Simple Linear Regression implemented from scratch with gradient descent.
- Part 2: Multiple Linear Regression on the California Housing dataset using 5-fold cross-validation and MSE reporting.
- Part 3: Gaussian Naive Bayes classifier implementation and evaluation.
- Part 4: Probability questions.

Main goal: practice building ML algorithms manually, evaluating with proper metrics, and interpreting results.
