# HW2 Autoencoders - Learning From Data

This homework focuses on: 

- Autoencoder (AE): building and training an encoder-decoder model for compressed latent representations and reconstruction.
- Variational Autoencoder (VAE): implementing probabilistic latent modeling with reconstruction + KL divergence objective.

Main files:

- homework_2.ipynb: notebook workflow for experiments and analysis.
- autoencoder.py: Autoencoder implementation.
- VAE.py: Variational Autoencoder implementation.
- mnist.npz: dataset 

Main goal: understand how latent-space learning works, compare deterministic AE and probabilistic VAE behavior, and evaluate reconstruction quality.
